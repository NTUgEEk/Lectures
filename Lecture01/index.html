Why Linux?

  • It's free!!!
  • Linux 相當適合用來開發
  • Linux 相當安全，不像 M$ Windows 要防備各種病毒的威脅
  • 會用 Linux 對找工作很有幫助XD

Linux everywhere

  • 你手上的 Android 正是 based on linux
  • Apple 的愛好者？Apple 系列的系統也是從 linux 的朋友 BSD 發展出來的
  • 你正看上網嗎？你所瀏覽的這些網站大都 架設於 linux 的伺服器上
  • 全世界的超級電腦有非常高 (99%?) 的比率 是採用 linux 系統

How to Use?

平時使用時會需要的大概有：

  • Browser - Firefox, Chrome
  • Input Method - gcin, fcitx, ibus
  • Terminal - gnome-terminal, Kconsole, terminator...
  • File Manager - Dolphin, Nautilus, Thunar...

或許你還會想要：

  • 音樂播放器 - Amarok, Clementine, DeaDBeeF...
  • FTP client - Filezilla
  • 影片播放器 - VLC, totem...
