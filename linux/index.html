Why Linux?

  • It's free!!!
  • Linux 適合用來開發程式和架設服務
  • Linux 一般來說相當安全，不太需要防備各種病毒的威脅
  • 會用 Linux 對找工作很有幫助 (現實面)

Linux everywhere

  • 你手上的 Android 正是 based on linux
  • 大部分的網路服候大都架設於 linux 的伺服器上
  • 全世界的超級電腦有非常高 (99%?) 的比率是採用 linux

Daily Use

平時使用時會需要的大概有：

  • Browser - Firefox, Chrome
  • Input Method - gcin, fcitx, ibus
  • Terminal - gnome-terminal, Kconsole, terminator...
  • File Manager - Dolphin, Nautilus, Thunar...

或許你還會想要：

  • 音樂播放器 - Amarok, Clementine, DeaDBeeF...
  • FTP client - Filezilla
  • 影片播放器 - VLC, totem...
