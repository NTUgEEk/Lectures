Repeat command

  • . : Repeat last command
  • <Number><Command> : 一次執行[Number]次[Command]

Examples:
5l : move left 5 characters
3aha<Esc> : insert hahaha

Delete, Yank & Paste

  • d : delete
  • dd : delete current line
  • y : yank
  • yy : yank current line
  • p/P : paste after/Before the cursor


Examples:
5dd : delete 5 lines
4yy : copy 4 line
dfa : delete to next character 'a'
y3l : copy 3 letter left

transitive verbs

vim中的指令大致上可以分成兩類
  • 不及物動詞: 按下後立刻執行
    ex: a,r,s
  • 及物動詞: 按下後可接修飾詞
    ex: d,y,c

position

  • movement: hjklggf...
  • Current Line: dd, yy...
  • range: a(: Around (), iw: in a word
Examples:
yaw: yank a word.
ci(: change inside ( )
di{: delete everything inside {}

Search

  • // ?: Forward/Backward search pattern
  • :s/[pattern1]/[pattern2]/[flag]: subsitute
    pattern1 with pattern2


Tips: pattern could be a regular expression!

regular expression

[abc] a or b or c
[^a] not a
\d digit (=[0-9])
. any character
\{x, y\} match [x, y] times
* match any times (=\{0, INF\})
\( ... \) group
\1 1st group
